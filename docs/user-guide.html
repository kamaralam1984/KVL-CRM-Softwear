App Overview & Navigation
CRM ka basic layout aur kaise navigate karein

Layout Structure

App teen main parts mein divided hai:

◀ Left Sidebar

Navigation menu jisme sab sections hain. Collapse kar sakte hain (sirf icons dikhenge). 4 groups: Main, Sales, Insights, System.

▲ Top Navigation Bar

Current section ka naam, search bar, dark/light mode toggle, aur AI Assistant button.

▶ Main Content Area

Jo bhi section select karo woh yahan dikhega. Smooth animation ke saath switch hota hai.

✦ AI Assistant Panel

Top-right ke AI button se open hota hai. Right side se slide in karta hai.

Sidebar Navigation Groups

GroupSectionsPurpose
MainDashboard, Leads, Customers, Deals, Tasks, CalendarRoz ka kaam
SalesPipeline, WhatsApp CRM, Email MarketingSales tools
InsightsTeam, Reports, Finance, Automation, AI InsightsData & analytics
SystemSettingsApp configuration

Sidebar Collapse

  1. Sidebar ke right edge par small arrow button hai.

  2. Click karo — sidebar sirf icons tak simat jaata hai (72px wide).

  3. Icons par hover karne se label tooltip dikhayi deta hai.

  4. Dobara click karo — full sidebar wapas aata hai (256px wide).

💡 Tip: Badges (jaise "12" Leads par, "5" Tasks par) unread ya pending items ka count dikhate hain.

Dashboard
Business ka complete snapshot ek jagah

Kya Dikhta Hai Dashboard Par

📈 KPI Cards (Top Row)

Total Revenue, Active Leads, Deals Closed, Conversion Rate — animated counters ke saath.

📉 Revenue Chart

12-month revenue trend — area chart. Hover karne par exact values dikhte hain.

🥧 Leads Funnel

Converted, Qualified, In Progress, Cold — pie chart mein lead distribution.

👥 Team Performance

Har team member ki deals aur revenue — bar chart comparison.

🕐 Recent Activity

Latest actions (calls, emails, deals) time ke saath.

💼 Top Deals

Highest value active deals with stage aur probability.

💡 Tip: Dashboard automatically load hone par KPI numbers 0 se actual value tak animate hote hain. Isko "animated counter" kehte hain.

Dashboard Se Kya Samjhein

  1. KPI cards dekh kar overall health samjhein — Revenue up hai ya down?

  2. Revenue chart mein monthly trend dekho — kaunsa mahina strongest tha?

  3. Leads funnel mein conversion rate check karo — zyada leads cold toh koi masla hai.

  4. Top deals mein priority deals note karo aur unpar action lo.

Leads Management
Potential customers ko track aur manage karo

Lead Status Types

StatusMatlabAction
🔥 HotHighly interested, ready to buyTurant follow-up karo
🌤 WarmInterested but needs nurturingRegular touchpoints
❄️ ColdLow interest or inactiveRe-engagement campaign

Lead Stages (Sales Funnel)

Discovery → Qualified → Proposal → Negotiation → Closed

Naya Lead Add Karna

  1. Upar right corner mein "+ Add Lead" button click karo.

  2. Name aur Company (required) fill karo.

  3. Email, Phone, Expected Value fill karo.

  4. Stage select karo (Discovery/Qualified/etc.)

  5. Status set karo (Hot/Warm/Cold).

  6. "Add Lead" button press karo — lead list mein add ho jayega.

Leads Filter Karna

  1. Top bar mein status buttons hain: All / Hot / Warm / Cold

  2. Click karne par sirf woh leads dikhenge.

  3. Stats cards automatically update ho jaate hain filtered data ke basis par.

💡 Tip: Lead Score (0-100) lead ki quality dikhata hai. 80+ = High priority. Score jitna zyada, utna jaldi contact karo.

Lead Card Mein Kya Dekho

Score Bar

Last Contact

"2h ago", "1d ago" — kab aakhri baar contacted tha

Tags

Industry/type labels jaise "Enterprise", "SaaS"

Value

Expected deal value in dollars

Naya Task Add Karna

  1. "+ Add Task" click karo.

  2. Task Title (required) aur Priority set karo.

  3. Due Date, Assignee (team member), Company name bharo.

  4. Save — task list mein appear ho jaayega.

Task Complete Karna

  1. Task ke left side par checkbox click karo.

  2. Task "completed" status mein aa jaayega (strikethrough text).

💡 Tip: Filter buttons use karo: All / Pending / In Progress / Completed — taaki focus bana raho.

Calendar
Meetings aur events schedule karo

Calendar Use Karna

  1. Left side par monthly grid dikhai deta hai. Blue dot wale days mein events hain.

  2. Right side par current month ke events listed hain.

  3. Month change karne ke liye ◀ ▶ arrows use karo (year bhi change hota hai automatically).

  4. Naya event add karne ke liye "+ Event" button click karo.

  5. Title, Day number, Time, Type (Meeting/Call/Team), Color select karo.

  6. Save — event calendar par appear ho jaayega.

Event Types

TypeIconUse Case
Meeting📹 VideoVideo calls, presentations

Sales Pipeline
Kanban board — deal stages visual track karo

5 Pipeline Stages

StageProbabilityMatlab
Prospect10%Naya potential lead
Qualified25%Interest confirm ho gayi
Proposal50%Proposal/demo diya
Negotiation75%Price/terms discuss ho rahi hain
Closed Won95%Deal almost confirmed

Pipeline Use Karna

  1. Har column ek stage represent karta hai. Deals unke stage mein cards ke roop mein dikhai deti hain.

  2. Column ke neeche "+ Add Deal" click karo — us stage ke liye deal add ho jaayegi.

  3. Total pipeline value har column ke header mein dikh rahi hoti hai.

  4. Deal cards par company name, value, aur owner dikh raha hota hai.

💡 Tip: Prospect column mein zyada deals ka matlab hai team naye leads generate kar rahi hai. Negotiation column heavy hona chahiye — woh close hone ke qareeb hain.

Naya Campaign Banana

  1. "+ New Campaign" button click karo.

  2. Campaign Name (required) enter karo.

  3. Status select karo: Draft / Scheduled / Sent.

  4. Date enter karo (optional).

  5. "Create Campaign" — campaign list mein add ho jaayega.

Quick Templates Use Karna

  1. Neeche "Quick Templates" section mein koi template click karo.

  2. Preview popup open hogi — template ka content dikh jaayega.

  3. "Use Template" click karo — campaign name pre-fill ho jaayega.

  4. New Campaign modal khulegi — customize kar ke save karo.

Member Card Features

  1. Har card par avatar, status dot (green=online, amber=away, gray=offline) dikhta hai.

  2. Crown icon (👑) top performer par hota hai.

  3. Card par hover karo — Phone aur Mail action buttons appear hote hain.

  4. Performance bar target ke against actual performance dikhati hai.

Naya Member Add Karna

  1. Grid ke last card par "+ Add Member" click karo.

  2. Full Name (required), Email, Role (select from list), Status bharo.

  3. Save — member grid mein appear ho jaayega.

💡 Tip: Performance 100%+ = target exceed kar liya. Green color = overachiever, Blue = on track, Amber = needs support.

Automation
Workflows automate karo

Workflow Card Features

  1. Har card par workflow name, trigger type, aur run count dikh raha hota hai.

  2. Play button (green) = workflow active hai. Click karne par pause ho jaata hai.

  3. Pause button (red) = workflow paused. Click karne par active ho jaata hai.

  4. Stats cards mein active workflows ki count auto-update hoti hai.

Naya Workflow Banana

  1. "+ New Workflow" click karo.

  2. Workflow Name aur Trigger (jaise "New Lead", "Invoice Overdue") bharo.

  3. Color select karo.

  4. Save — workflow list mein add ho jaayega.

Profile Edit Karna

  1. Profile card mein "Edit Profile" button ya Profile Settings expand karke "Edit Profile →" click karo.

  2. Name, Role, Email, Phone edit karo.

  3. "Save Changes" — changes turant profile card mein reflect hote hain.

Settings Items Expand Karna

  1. Koi bhi settings item click karo (jaise "Notifications").

  2. Smooth animation se detail view expand ho jaata hai.

  3. Doosra item click karo — pehla automatically collapse ho jaata hai.
